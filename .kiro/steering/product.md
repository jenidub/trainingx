# Product Overview

AI prompting education platform that teaches users how to effectively work with AI systems through hands-on practice, assessments, and skill tracking.

## Core Features

- **Practice Zone**: Leveled challenges (Beginner → Intermediate → Advanced) with real-world scenarios
- **Placement Test**: Initial skill assessment to personalize learning path
- **Daily Drills**: Spaced repetition system for skill retention
- **Dashboard**: Track prompt score, skills, badges, and progress
- **Career Matching**: AI-powered career recommendations based on skills
- **Community**: Posts, discussions, and peer learning

## User Journey

1. Sign up → Placement test → Personalized track recommendation
2. Complete daily drills and practice projects
3. Earn badges, improve prompt score, unlock levels
4. Get career matches and learning path recommendations
